## Commands

### Create (default)

```
/claw-config-skill-creator <new-skill-name>
```

Outputs a proposal: file tree + full contents.

### Review

```
/claw-config-skill-creator review <skill-path-or-name>
```

Outputs a checklist-based audit.

## Notes

- Create mode is proposal-only.
- Review mode should reference the same checklists used in creation.
