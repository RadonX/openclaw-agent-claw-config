# Archetypes (common claw-config skill types)

A creator skill should generate different skeletons depending on intent.

## 1) Routing / bindings refactor

Must include:
- before/after coverage map
- activation gate audit for touched peers
- minimal patch + verification

## 2) Telegram channel behavior

Must include:
- mention/allowlist/topic overrides model
- test plan (DM + group + topic)

## 3) Exec / approvals / elevated debugging

Must include:
- layered diagnosis model (host/security/allowlist/elevated)
- doc pointers

## 4) Incident playbook

Must include:
- log locations
- minimal reproduction
- rollback + postmortem note
